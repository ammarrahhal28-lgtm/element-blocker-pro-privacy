Privacy Policy – Element Blocker Pro

Element Blocker Pro is a browser extension designed to help users hide unwanted elements on websites they visit. Your privacy is extremely important to us, and this policy explains what data is (and is not) collected by the extension.

1. Information We Do Not Collect

The extension does not collect, store, or transmit any personally identifiable information (PII), browsing history, financial information, passwords, authentication details, or any sensitive data.

2. Local Storage Only

All user preferences, blocked items, and extension settings are stored locally on your device only using Chrome's storage API. No information is sent to external servers.

3. Permissions Explained

activeTab: Allows the extension to temporarily access the currently opened tab so the user can hide selected elements.
scripting: Required to inject the content script that highlights and removes elements.
<all_urls>: Needed to allow the user to hide elements on any website they choose. No data is collected from these websites.

4. No Tracking or Analytics

The extension does not use analytics tools, trackers, or remote logging.

5. No Third-Party Data Sharing

We do not share, sell, or transmit any user data to third parties.

6. Changes to This Policy

If this privacy policy is updated, the new version will be available at the same URL.
